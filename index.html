About Me

Halo! Nama saya Galuh Dipa Bharata, saat ini saya sedang berkuliah di Institut Teknologi Bandung, dengan program studi Sistem dan Teknologi Informasi. Selama saya berkuliah, saya menjalankan beberapa project, seperti merakit IoT, membuat web sederhana, dan bahkan mengintegrasikan keduanya. Hal ini tentu merupakan tantangan bagi saya, yang mana saya baru saja memulai belajar untuk menuliskan kode ketika memasuki bangku kuliah. Tapi hal tersebut tidak menjadi hambatan saya dalam belajar.
